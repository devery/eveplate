Our Ultimate Goal

Eveplate is a generator to help you to kickstart new projects based on the open-source protocol and javascript framework that has been built and developed by Devery.io.

The generator acts as a plugin that can be run through command line to scaffold complete projects or useful parts.

It is a complex clientside-stack to ensure seamless workflow comprising tools and frameworks to allow developers to build unique verification Dapps based on Devery’s framework technology.

We aim to provide everything needed to get your projects started by helping to eliminate roadblocks during a manual setup.

This shows Devery’s approach for providing comprehensive and simple tools backed with full documentation to ensure seamless development without the need for direct interaction with the blockchain.

This contribution into Devery’s decentralized ecosystem will improve the building process and make it even easier.

Our Cutting-Edge Tools

Devery.js is an open source javascript framework developed by Devery.io to ensure simple building verification apps based on an open source protocol which negates the need of direct interaction with the Etherium blockchain. Devery JS library provides all the writing and reading functions to ensure seamless workflow and to allow extending basic functionality.

The Ethereum network is an open source, public, blockchain-based distributed computing platform and operating system featuring smart contract (scripting) functionality. On Ethereum, you can write code that controls digital value, runs exactly as programmed, and is accessible anywhere in the world.
